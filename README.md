## Overview

**Playora** is a demo kids' website that brings together learning and play in a vibrant, engaging interface. Designed with young users in mind, the site features a child-friendly layout with bright visuals, intuitive navigation, and interactive elements — all built from scratch using **vanilla HTML, CSS, and JavaScript** with no external frameworks.

This project demonstrates front-end design fundamentals including responsive layout, UI/UX thinking for a specific audience, and clean, accessible web structure in a single self-contained file.

---

## Features

- **Kid-Friendly UI** — Bright colors, large fonts, and playful design elements
- **Easy Navigation** — Simple, intuitive layout suitable for young users
- **Interactive Elements** — JavaScript-powered interactivity and animations
- **Responsive Design** — Adapts to different screen sizes
- **Sections for Activities** — Organized content for games, learning, or exploration
- **Zero Dependencies** — Pure HTML/CSS/JS, runs in any browser instantly

## Getting Started

No installation or setup needed — just open the file in a browser!

```bash
# Clone the repository
git clone https://github.com/jasvi05/Playora_Kids_Website_Demo.git
cd Playora_Kids_Website_Demo

# Open in browser
open index.html
# or double-click index.html in your file explorer
```

> Works in all modern browsers — Chrome, Firefox, Safari, Edge.

---

## Customization Guide

Since everything is in a single file, customization is straightforward:

```html
<!-- Change colors in the <style> section -->
:root {
  --primary-color: #FF6B6B;   /* Change to your preferred color */
  --accent-color: #FFD93D;
  --bg-color: #F9F9F9;
}

<!-- Add new sections in the <body> -->
<section class="new-section">
  <h2>New Activity</h2>
  <!-- your content -->
</section>
```
